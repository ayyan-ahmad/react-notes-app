# 📝 Notes App

## 📌 Description
A clean and responsive Notes App built using React. Users can easily add and delete notes.

## 🚀 Features
- Add Notes
- Delete Notes
- Simple and clean UI

## 🛠️ Tech Stack
- React
- JavaScript
- Tailwind CSS

## ▶️ How to Run
git clone https://github.com/ayyan-ahmad/react-notes-app.git
cd react-notes-app
npm install
npm run dev
